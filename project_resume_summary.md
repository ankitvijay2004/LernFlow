**LearnFlow LMS (MERN Stack)**
**Tech Stack**: MongoDB, Express.js, React.js, Node.js, Vite, JWT, REST API, Render, Lucide React, Glassmorphism, CSS3
**Key Responsibilities & Achievements**:
-   **Full-Stack Development**: Built a comprehensive Learning Management System (LMS) from scratch using the MERN stack.
-   **Advanced UI/UX**: Designed a premium, responsive interface featuring Glassmorphism, 3D hover transformations, and smooth micro-animations.
-   **Secure Authentication**: Implemented robust user authentication and role-based access control (Student/Instructor) using JWT and bcrypt.
-   **Dynamic Theming**: Engineered a global light/dark theme engine with persistent user preferences and optimized CSS variable transitions.
-   **Search & Filtering**: Developed real-time search functionality allowing users to filter courses by title and category seamlessly.
-   **Security Hardening**: Protected the application against common vulnerabilities using Helmet (secure headers), NoSQL injection sanitization, and API rate limiting.
-   **Cloud Integration**: Configured MongoDB Atlas for scalable data storage and implemented a unified build process for streamlined Render deployment.
-   **Functional Modules**: Built dedicated dashboards for instructors to manage courses and a distraction-free lesson player for students.
